#　 BeautifulSoup
BeautifulSoup を利用したサンプルコード

## install

```
$ pip3 install beautifulsoup4
$ pip3 install lxml
```

Web スクレイピングで利用するので入れていなければ install

```
$ pip3 install requests
```
